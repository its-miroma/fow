# How to Install: [Prism Launcher](<https://prismlauncher.org/>)

1. Download and install [Java](<https://prismlauncher.org/wiki/getting-started/installing-java>)
2. Open the **Prism Launcher**
3. Click on the **Modrinth** tab on the left
4. Click on **Fabulously Optimized**
5. If you do not want the latest version, select the version you want from the dropdown.
6. Click on **OK**. Once installed, you should see something <!-- TODO: what though? -->
7. Double-click the icon to play. Minecraft should open up
8. If you can see `Fabulously Optimized` in the bottom-right corner, you're done!
