# How to Install: [Modrinth App](<https://modrinth.com/app>)

{% hint style="warning" %}
Modrinth App version 0.6.2 or higher is required
{% endhint %}

1. Open the **Modrinth App**
2. Open the **Popular packs** section
3. Hover over `Fabulously Optimized` tile
4. Click on the Install button. <!-- TODO: Emoji --> Once installed, you should see a new instance under "Jump back in"
5. If you do not want the latest version, click on the tile, then on `Versions`, then on the download button <!-- TODO: Emoji -->
6. Hover over the newly created instance
7. Press the Play button. <!-- TODO: Emoji --> Minecraft should open up
8. If you can see `Fabulously Optimized` in the bottom-right corner, you're done!
