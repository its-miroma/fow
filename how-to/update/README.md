# How to Update

{% hint style="danger" %}
**You should ALWAYS make backups before updating**. As with any other piece of software, things may go wrong when updating, for many reasons. In the case of FO, this includes, but is not limited to: the launcher, the hardware, your custom configuration, the mods you added, or (rarely) because of FO.
{% endhint %}

{% hint style="danger" %}
FO only provides support for updating to stable versions. We do NOT support updating from a stable to an unstable (alpha, beta) version, nor do we support downgrading. In those cases, consider creating a separate installation/instance.

You should never update mods bundled in FO manually. The modpack is carefully crafted together to ensure full compatibility between its mods, so updating a mod individually may cause conflicts with the others. Of course, if you've [added custom mods to FO](/how-to/add-mods/README.md), you must update those on your own.
{% endhint %}

{% hint style="info" %}
If you want the recommended FO configuration, you may [get the latest recommended settings](/info/options.md)
{% endhint %}

{% hint style="info" %}
If a server you're playing on is out-of-date, the [ViaFabricPlus mod](https://modrinth.com/mod/viafabricplus) might help you.
{% endhint %}
