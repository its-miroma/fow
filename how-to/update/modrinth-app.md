# How to Update: [Modrinth App](<https://modrinth.com/app>)

1. Open the **Modrinth App**
2. Select the existing FO instance
3. Click on the **Update modpack** button on the top-right
4. If you do not want the latest version, select the version you want from the dropdown
5. Click on the **two-arrows** button. The modpack should get updated
6. Once updated, launch the instance. Minecraft should open up
7. If you can see the newly installed version in the bottom-right corner, you're done!
