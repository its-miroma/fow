# Resource Packs

FO bundles some resource pack for better experience and clarity, and enables some mod-provided resource packs.

The resource packs use minimal resources, they are highly compatible, and they can be [translated to your language](../../about/contribute/translate.md).

- [Fast Better Grass](fast-better-grass.md)
- [Mod Menu Helper](mod-menu-helper.md)
