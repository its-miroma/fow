# mod-menu

It also adds icons showing configurable mods: if **`🛠️`** or **`✏️`** is blue, you can configure the mod by clicking the config button

![config](https://i.ibb.co/j35cBtn/image.png)

* If you **don't see any icons** and you didn't add the mod yourself, you don't have the Mod Menu Helper resource pack enabled for some reason. If you can't see `🛠️` or `✏️`, then MMH is disabled
   1. Click on `Done`
   2. Click on `Options...`
   3. Click on `Resource Packs...`
   4. Click on `⏵` next to **Mod Menu Helper.zip**
   5. Click on `Done`
